## About The Project

This project was a weekly assignment for the MITx Pro Emeritus Full Stack Developer course. The assignment was to refactor a shopping cart that could "restock" from an API created with strapi

<p align="right">(<a href="#readme-top">back to top</a>)</p>





<!-- GETTING STARTED -->
## Getting Started

Create a fork of this Repository

### Installation


1. Clone the repo
 
2. Install NPM packages
 
3. Start a local server and run the files
